### **Monitoring Stage Progress:**
1. Scroll to **Stage-by-Stage Progress** section
2. Each stage shows:
   - Numbered badge (1-6)
   - Stage name
   - Completion percentage
   - LGU count (completed / total)
   - Progress bar with color coding
   - Status breakdown
   - Timeline information

### **Identifying Bottlenecks:**
1. Look for stages with **low completion rates** (red/yellow)
2. Check **"Not Started"** counts - these need attention
3. Compare **"In Progress"** vs **"Completed"** ratios
4. Review timeline info to see if stage is taking too long

### **Tracking Timeline:**
1. Note **"First completed"** date for each stage
2. Compare with **"Last completed"** date
3. Large gaps indicate uneven progress across LGUs
4. Use this to identify which LGUs need support
